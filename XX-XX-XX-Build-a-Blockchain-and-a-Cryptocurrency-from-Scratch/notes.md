# Notes
## Initialise project
```bash
npm init -y
```

## Install Nodemon as a dev dependency
```bash
npm i nodemon --save-dev
```

## SHA 256
SHA 256 is a one-way hash that produces an unique 32 byte (256 bit) hash value for unique inputs.

## Install CryptoJS
```bash
npm i crypto-js --save
```

npm i jest --save-dev

npm i express --save
